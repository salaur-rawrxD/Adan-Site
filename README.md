# Adan Aispuro Studio Site

A polished one-page consulting site for product strategy, digital systems, and operator-focused business improvement.

## Stack

- Next.js App Router
- React
- Tailwind CSS
- Framer Motion
- lucide-react

## Local Development

```bash
npm run dev
```

Open `http://localhost:3000`.

## Checks

```bash
npm run lint
npm run build
```
